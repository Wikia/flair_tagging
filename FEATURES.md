# The Wizarding Forums — Feature Overview

## Current Features

### Category Tabs
Three top-level tabs filter threads by type:
- **FanFic** — fan-written stories and fiction
- **Trade** — item trading and exchange posts
- **Opinion** — discussion and debate threads

Each tab has a distinct house-inspired color (Gryffindor red, Slytherin green, Ravenclaw blue).

### Keyword Search
A search bar within each category filters threads in real time by matching the keyword against both the thread **title** and **content**.

### Sort Pills
Four inline sort options let users reorder the thread list:
- **Newest** — most recently posted first
- **Oldest** — earliest posted first
- **Most Liked** — threads with the most liked replies first
- **Most Replies** — threads with the highest reply count first

### Thread Cards
Each thread is displayed as a card showing:
- Author avatar (initial letter), name, and post date
- Thread title and a short content excerpt
- Reply count and like count

### Thread Detail View
Clicking a thread card opens a full detail view showing:
- Complete thread content, author, date, and category badge
- All replies in chronological order, each showing author, timestamp, and content
- Liked replies are highlighted with a gold left border and a "Liked" badge

### Harry Potter Themed UI
- Dark midnight starfield background fading into warm parchment tones
- Gold and parchment color palette throughout
- *IM Fell English* serif font for an aged, magical feel
- Subtle moon glow and scattered stars in the background

## Planned Feature

### Wiki Filtering
Users will be able to select one or more **wikis** (e.g. Harry Potter, Fantastic Beasts, Hogwarts Legacy) to scope the thread list. This works on top of the existing category tabs — for example, a user interested only in Hogwarts Legacy discussion can select that wiki and then browse FanFic, Trade, or Opinion threads specific to it.

**Proposed behaviour:**
- A wiki selector appears above the category tabs, styled as a dropdown or a horizontal pill group
- Selecting a wiki filters all threads to those tagged with that wiki
- Multiple wikis can be selected simultaneously to broaden the view
- The "All Wikis" default shows threads from every wiki, preserving current behaviour
- Each wiki can define its own set of relevant categories if needed in future
